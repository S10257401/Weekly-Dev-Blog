Week 14

    • Lottie is a JSON-based animation file format that enables animations on any platform easily.
    • It allows additional interactivity on websites or mobile platforms.
    • Lottie files are small, lightweight, and work on any device without pixelation.
    • Lottie was first created by AirBnB.
    • File size matters: Lottie animations are extremely lightweight because they are essentially code.
    • In the Lottie workflow, you work closely with SVG or After Effects.
    • Lottie can be imported and used in several platforms, but in your scenario, you are using it for the web version.
    • Working with LottieFiles involves finding a suitable animation, importing the JSON code into your HTML file, and optionally using the LottieEditor to modify the animation.
