Week 3

  • Learned about the basic structure of HTML documents starting with a boilerplate.
  • Explored additional HTML elements including list items and the <div> element for containerization.
  • Studied the <a> element for creating hyperlinks with the href attribute, which can link to internal or external resources and be configured to open in different ways using the target attribute.
  • Explored the importance of the alt attribute for images, providing alternate text for accessibility and the title attribute for displaying text upon mouse hover.
  • Learned about HTML attributes, which add additional information to HTML tags, and practiced incorporating them into tags.
  • Practiced inserting images into HTML documents using the <img> element, understanding relative and absolute paths for image sources.
  • Understood the significance of proper file naming conventions in web development, including using lowercase, meaningful file names, replacing spaces with dashes, and ensuring proper file extensions for compatibility and organization.
